## Installation guide

```sh
# clone the project
git clone https://github.com/jurerotar/

# Install dependencies
npm install

# Starts a development server with hot-reloading
npm run dev

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint
```
